# qa-portfolio-projects

## Sobre
Este repositório reúne projetos práticos de testes de software desenvolvidos com base em um sistema de e-commerce.

## Projetos incluídos
- Documentação de produto
- Técnicas de testes
- Mapa de risco
- Fluxo de trabalho ágil

## Projetos em destaque
- Bug Report
- Gherkin (BDD)
- User Stories

## Objetivo
Demonstrar na prática conhecimentos em QA, incluindo testes manuais, análise de qualidade e metodologias ágeis.
